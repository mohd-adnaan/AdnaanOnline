## 🚀 Quick Start

**Prerequisites:** Node.js 16+ and npm/yarn installed. That's it.

```bash
# Clone this bad boy
git clone https://github.com/mohd-adnaan/AdnaanOnline.git
cd AdnaanOnline

# Install dependencies
npm install

# Fire it up
npm start

# Open http://localhost:3000 and watch the magic happen ✨
```

### 🐳 Or Use Docker (if you're fancy)

```bash
docker build -t adnaan-portfolio .
docker run -p 3000:3000 adnaan-portfolio
```
